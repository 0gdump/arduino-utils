# Chargen

## About

Simple painting app for drawing characters for LCD displays

## Usage

Run:

```
chardraw [CHAR NAME]
```

Window:

```
┌────────────────────────────────────┐
│              Chardraw              │
┌────────────────────────────────────┐
│            ┌──────────┐            │
│            │░░██████░░│            │
│            │██░░░░░░██│            │
│            │██░░██░░██│            │
│            │██░░██░░██│            │
│            │██░░██░░██│            │
│            │██░░░░░░██│            │
│            │░░██████░░│            │

└────────────────────────────────────┘
```

Sample output:

```c
byte char[8] = {
	B01110,
	B10001,
	B10101,
	B10101,
	B10101,
	B10001,
	B01110,
};
```
