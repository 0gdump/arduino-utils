# Arduino Utils

Utilities and scripts for automating typical Arduino programming tasks.

## Interactive Utils

### Chardraw

Simple painting app for drawing characters for LCD displays. [Read more](https://github.com/0gdump/arduino-utils/tree/main/chardraw)

## Static Utils

### Chargen

Character declaration generator from 5x7 BMP images for LCD displays. [Read more](https://github.com/0gdump/arduino-utils/tree/main/chargen)
